📱 서원 시상식 앱 접속 방법

===========================================
🔍 검색 엔진에서 찾는 방법
===========================================

⚠️ 주의: 이 앱은 로컬 네트워크에서만 작동하므로
구글이나 크롬 검색 엔진에서는 찾을 수 없습니다.

하지만 더 쉬운 방법들이 있습니다!

===========================================
✅ 방법 1: 북마크 저장 (가장 추천!)
===========================================

1. 컴퓨터에서 앱에 접속
2. 브라우저 북마크에 저장 (Cmd+D 또는 Ctrl+D)
3. 북마크 이름을 "서원 시상식"으로 설정
4. 나중에 북마크에서 바로 클릭!

모바일에서도:
- 접속 후 브라우저 메뉴 → 북마크 추가
- 또는 홈 화면에 추가 (더 편함!)

===========================================
✅ 방법 2: 홈 화면에 추가 (모바일)
===========================================

iPhone:
1. Safari에서 접속
2. 공유 버튼(□↑) 클릭
3. "홈 화면에 추가" 선택
4. 앱처럼 사용 가능!

Android:
1. Chrome에서 접속
2. 메뉴(⋮) → "홈 화면에 추가"
3. 앱처럼 사용 가능!

===========================================
✅ 방법 3: QR 코드 사용
===========================================

1. 홈 페이지에서 "QR 코드 보기" 클릭
2. 폰 카메라로 스캔
3. 자동으로 접속!

===========================================
✅ 방법 4: 링크 공유
===========================================

1. 홈 페이지에서 링크 복사
2. 카카오톡/문자로 자신에게 보내기
3. 링크 클릭해서 접속

===========================================
💡 팁: 간단한 이름으로 접속
===========================================

브라우저 주소창에 입력:
- "서원 시상식" (검색 후 북마크 클릭)
- 또는 직접 IP 주소 입력

===========================================
📌 현재 접속 주소
===========================================

컴퓨터: http://localhost:3000
폰/다른 기기: http://192.168.219.106:3000

(터미널에 표시된 Network 주소를 사용하세요)

===========================================
🎯 가장 쉬운 방법
===========================================

1. 한 번 접속해서 북마크 저장
2. 또는 홈 화면에 추가
3. 그 다음부터는 북마크/홈 화면에서 바로 접속!

이렇게 하면 검색할 필요 없이 바로 접속 가능합니다! 🚀
