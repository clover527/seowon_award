# 🚀 빠른 시작 가이드

## 서버 실행 방법

### 방법 1: 한 번에 실행 (권장)
```bash
npm run dev:all
```

이 명령은 프론트엔드와 백엔드를 동시에 실행합니다.

### 방법 2: 개별 실행

**터미널 1 - 백엔드 서버:**
```bash
npm run dev:server
```

**터미널 2 - 프론트엔드:**
```bash
npm run dev
```

## 접속 방법

### 컴퓨터에서 접속
브라우저에서 자동으로 열립니다. 또는 터미널에 표시된 주소로 접속하세요.

### 휴대폰에서 접속

#### 방법 1: QR 코드 사용 (가장 쉬움!)
1. 컴퓨터 브라우저에서 `http://[당신의-IP]:3000/qr.html` 접속
   - 예: `http://192.168.219.106:3000/qr.html`
2. 표시된 QR 코드를 폰 카메라로 스캔
3. 자동으로 접속됩니다!

#### 방법 2: 링크 복사
1. 터미널에 표시된 **Network** 주소 확인
   - 예: `http://192.168.219.106:3000`
2. 이 주소를 폰에 복사해서 접속
3. 카카오톡이나 문자로 자신에게 보내면 됩니다

#### 방법 3: 북마크 저장
1. 컴퓨터에서 접속한 후 북마크 저장
2. 북마크를 클라우드에 동기화
3. 폰에서도 같은 북마크 사용

## 주의사항
- ✅ 컴퓨터와 폰이 **같은 Wi-Fi 네트워크**에 연결되어 있어야 합니다
- ✅ 컴퓨터의 방화벽이 허용되어야 합니다 (Mac에서 처음 실행 시 팝업에서 허용)
- ✅ 서버가 실행 중이어야 합니다

## 문제 해결

### 연결이 안 될 때
1. 같은 Wi-Fi에 연결되어 있는지 확인
2. 서버가 실행 중인지 확인
3. 방화벽 설정 확인
4. 터미널에 표시된 정확한 IP 주소 사용

### QR 코드가 안 나올 때
브라우저에서 직접 접속해도 됩니다. 터미널에 표시된 Network 주소를 사용하세요!
